IDEA 常用快捷键

  • alt + Insert ：生成 get/set 方法
  • Ctrl + Alt + t ： 使用 try/catch
  • Ctrl+Alt+Left/Right ：光标定位到上一个/下一个浏览位置处
  • Ctrl+Shift+Backspace：光标定位到上次修改的地方
  • Ctrl+Alt+L：格式化代码
  • Ctrl+Alt+O：去除未使用的 import 包或类
